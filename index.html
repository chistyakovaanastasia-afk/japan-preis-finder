Einstellungen

Beide findest du bei deiner App auf webservice.rakuten.co.jp (Application ID + Access Key). Ohne beide funktionieren die Shop-Links unten weiterhin. Mit beiden zeigt das Tool oben automatisch den günstigsten Rakuten-Preis + Kauflink an.

🔒 Beide Werte werden nur in diesem Browser gespeichert (localStorage), nie an Dritte gesendet und nicht im Code abgelegt. Nutze das Tool nicht auf fremden Geräten. Den Access Key kannst du bei Rakuten jederzeit neu erzeugen.
